# Terminal Emocional

> **Rituales digitales para procesos interiores.**

Un proyecto web conceptual y narrativo diseñado para simular una terminal de sistema que ejecuta la desinstalación de patrones de comportamiento (como el *masking*, la hipervigilancia o las respuestas de trauma). 

Ha sido diseñado específicamente como una experiencia inmersiva y estética, optimizada para su grabación y visualización en redes sociales en formato vertical (mobile-first, 720x1280).

### 🚧 Estado del Proyecto
Este proyecto **sigue en desarrollo**. Actualmente incluye las secuencias de desinstalación del "Masking" en español e inglés, con animaciones cíclicas y diseño ajustable.
